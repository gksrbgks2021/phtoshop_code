# About The Project
---
### PyQt5 , Open-CV 를 사용한 포토샵 앱


### 기능 나열
- 배열 연산 이용한 회전 , 좌우반전
- 마우스 오버레이 시 UI 움직이는 기능
- RGB 트랙바
- 반전 필터
- salt and peper 기법을 사용한 점묘법 필터
- 이미지 자르기, 샤프닝, 블러링 기능
